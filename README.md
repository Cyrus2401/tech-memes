# Tech Memes

A lightweight **web-based community dashboard designed for sharing tech humor and interacting with tech memes**.

---

## Description

Tech Memes is a modern web application designed for sharing tech-oriented humor and IT culture. It allows administrators to publish memes, while users can browse the feed, copy share links, download images directly, and interact via a database-driven persistent likes system.
